📂 채용 데이터 업로드 도구

전임교원 채용 RAWDATA 엑셀 파일을 업로드하면 대시보드용 JSON을 자동 생성합니다

1

엑셀 파일 선택

채용 RAWDATA 엑셀 파일(.xlsx)을 드래그하거나 클릭해서 선택하세요.
4행이 헤더(학년도, 학기, 채용구분...)인 파일을 사용하세요.

2

시트 선택 후 JSON 생성

RAWDATA가 포함된 시트를 선택하고 버튼을 누르면 recruitment.json이 생성됩니다.

3

파일 배포

다운로드한 recruitment.json을 data/ 폴더에 넣고 GitHub에 업로드하면 즉시 반영됩니다.
