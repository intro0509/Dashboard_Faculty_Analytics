📂 데이터 업로드 도구

전임교원 현황표 엑셀 파일을 업로드하면 대시보드용 JSON을 자동 생성합니다

1

엑셀 파일 선택

RAWDATA 시트가 포함된 .xlsx 또는 .xlsm 파일을 드래그하거나 클릭해서 선택하세요.

2

확보율 입력 (선택)

PIVOT 시트에 있는 확보율 데이터를 입력하세요. 생략하면 이전 데이터가 유지됩니다.

3

JSON 생성 및 서버 업로드

생성된 dashboard_data.json을 서버의 dashboard.html과 같은 폴더에 올리면 즉시 반영됩니다.

엑셀 파일을 여기에 드래그하거나 클릭하세요

📈 전임교원 확보율 입력
